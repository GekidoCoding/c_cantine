# Application Mobile de Gestion des Repas des Employés

## Description générale

L’**application mobile de gestion des repas des employés** est une application développée avec le framework **Ionic**, destinée à faciliter l’organisation, le suivi et la gestion des repas au sein d’une entreprise.

Elle permet aux employés et aux responsables de consulter et gérer les informations liées aux repas de manière simple, rapide et accessible depuis un appareil mobile.

---

## Fonctionnalités principales

L’application mobile permet notamment :

* La gestion des **repas des employés** (consultation, suivi, historique)
* La visualisation des menus disponibles
* Le suivi des consommations de repas par employé
* Une interface mobile intuitive et ergonomique
* Une interaction fluide avec un système backend via des API

---

## Utilisateurs de l’application

L’application est destinée à :

* **Les employés** : pour consulter les repas, menus et informations associées
* **Les responsables / administrateurs** : pour gérer les données liées aux repas et au personnel

---

## Architecture

L’application repose sur une architecture **client – serveur** :

* **Client mobile** : Application Ionic (Android / iOS)
* **Backend** : API REST (à préciser selon l’implémentation)
* **Échanges de données** : via des API sécurisées

---

## Technologies utilisées

* **Framework mobile** : Ionic /Angular
* **Langage** : TypeScript / JavaScript
* **Architecture** : Application mobile hybride
* **Backend** : API REST

---

## Objectif du projet

L’objectif de cette application est de **simplifier la gestion des repas des employés**, d’améliorer l’organisation interne et d’offrir une solution mobile moderne adaptée aux besoins de l’entreprise.
